📐 Vertical ROM Accuracy Test

Validate IMU vertical displacement against known distances. Export results to CSV.

📋 How to Test

  1. Connect your ESP32 IMU via Bluetooth.
  2. Rest Drift Test: Place sensor flat, record 10s — drift should be near 0 cm.
  3. Known Distance Test: Set a target distance (e.g. 20 cm), calibrate baseline, lift sensor to that height, hold, lower back down.
  4. Repeat multiple trials. The table below logs each trial with measured peak vs. expected distance and error %.
  5. Export CSV with all raw samples + trial results for offline analysis.
